📌 Project Overview

This project analyzes a telecom customer churn dataset using Python. The main objective is to understand customer behavior and identify factors that influence customer churn.

The project uses Pandas for data analysis and Matplotlib for data visualization.

🚀 Features
Load and explore the telecom customer churn dataset.
Display dataset shape and column information.
Preview the first five rows of the dataset.
Analyze customer churn distribution.
Compare contract types with customer churn.
Calculate average monthly charges based on churn status.
Visualize the results using bar charts.
🛠️ Technologies Used
Python
Pandas
Matplotlib
📂 Dataset

The project uses the Telco Customer Churn Dataset.

Dataset file:

WA_Fn-UseC_-Telco-Customer-Churn.csv
📊 Analysis Performed
1. Dataset Exploration
Dataset shape
Column names
First five records
2. Churn Distribution
Count of customers who stayed and left
3. Contract Type vs Churn
Relationship between customer contract types and churn
4. Monthly Charges Analysis
Average monthly charges for churned and non-churned customers

▶️ How to Run
Clone the repository.
Install the required libraries:
pip install pandas matplotlib
Place the dataset file in the project folder.
Run the Python script:
python main.py
